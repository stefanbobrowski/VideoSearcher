# VideoSearcher Frontend

This is the frontend for the VideoSearcher project, built with React, TypeScript, and Vite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build:
   ```bash
   npm run preview
   ```

## SCSS Support

SCSS modules are supported. Example: `About.module.scss`.

## Linting & Formatting

- ESLint is configured for TypeScript and React.
- Prettier is used for code formatting. Run `npm run format` to format code.

## Deployment

A sample `cloudbuild.yaml` is provided for Google Cloud Build integration.

## Other Files

- `robots.txt` is in the `public/` folder for SEO and crawler control.
- `.gitignore` is set up to ignore build and dependency files.
